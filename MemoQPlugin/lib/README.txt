Place MemoQ SDK DLLs here:

Required files:
- MemoQ.PreviewInterfaces.dll
- MemoQ.Addins.Common.dll

These files can be obtained from:
1. MemoQ installation directory (usually C:\Program Files\MemoQ\)
2. MemoQ SDK download (https://www.memoq.com/developer)

After placing the DLLs, run the build script:
  .\build.ps1
